# 🎥 YouTube Downloader 🚀

Un **script Python** pratique pour télécharger vos vidéos et playlists YouTube préférées directement sur votre ordinateur. 🎉

---

## 🛠️ Prérequis

- 🐍 **Python 3.7** ou version supérieure.
- 📦 **pip** (inclus avec Python).

---

## ⚙️ Installation

1. Clonez ce dépôt ou téléchargez les fichiers nécessaires. 📂
2. Exécutez le script `setup.py` pour installer tout ce qu'il faut. 🛠️

```bash
python setup.py
````

✨ **Ce que fait le script :**

* Installe les dépendances nécessaires. 📥
* Crée un dossier `Youtube download` dans vos `Téléchargements`. 📁
* Ajoute un fichier `url.txt` avec une URL d'exemple. ✏️

---

## 🚀 Utilisation

1. Lancez le script principal :

```bash
python Downloader.py
```

2. Suivez les instructions affichées dans le terminal. 🖥️

### 🌟 Options disponibles :

* **Option 1 :** Saisissez une URL pour télécharger une vidéo YouTube. 🔗
* **Option 2 :** Utilisez un fichier `url.txt` pour télécharger plusieurs vidéos. 📜
* **Option 3 :** Saisissez une URL pour télécharger une playlist entière. 📋

### ⚙️ Paramètres supplémentaires :

* **Qualité vidéo :** Choisissez entre `high` ou `low`. 🎞️
* **Téléchargement audio uniquement :** Sélectionnez `yes` ou `no`. 🎵

---

## 📂 Où trouver vos téléchargements ?

* Les vidéos/audios sont enregistrés dans :
  `Téléchargements/Youtube download/data`. 📥

* Vous pouvez éditer le fichier `url.txt` pour ajouter d'autres liens. ✏️

---

## 🔧 Désinstallation

Après l'installation initiale, vous pouvez supprimer `setup.py` :

```bash
rm setup.py
```

---

## 📦 Dépendances

Pas de souci ! Elles sont installées automatiquement via `setup.py` :

* 📜 `yt-dlp`
* 🐍 `pytube`
* 📶 `tqdm`
